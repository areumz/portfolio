## :pushpin: Intro
만들고 싶은 것들에 대한 아이디어가 자주 머릿속에 떠다니는데, 막상 구현하면서 생기는 갭에 때로는 답답함도 느꼈습니다.
하지만 문제가 발생할 때 치열하게 고민하고 끈질기게 파고들어 해결하고 나면 그 기쁨은 이루 말할 수 없었습니다.
부족함 앞에서 포기하지 않고, 꾸준히 배우고 학습하며 성장하는 그런 프론트엔드 개발자가 되고 싶습니다.
